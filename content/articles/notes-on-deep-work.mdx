## Make the Task Smaller

Focus often returns when the task becomes visible. Instead of writing "learn web development", write "build one button and make it responsive".

## Use a Landing Pad

Keep one scratch note open. When another idea appears, put it there instead of chasing it.

<PullQuote>
The point is not perfect focus. The point is returning gently.
</PullQuote>

## Try a 25-Minute Shape

Set one intention, remove one distraction, and work for 25 minutes. Then stop. The stop matters because it teaches the brain that focus has an exit door.

## A Small Review

At the end, write three lines:

1. What did I finish?
2. What blocked me?
3. What is the next visible step?
